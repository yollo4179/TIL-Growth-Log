MyBatis란 ❓

- `MyBatis`는 자바 기반의 애플리케이션에서 DB에 쉽게 접근할 수 있도록 도와주는 `SQL Mapper` `프레임워크`입니다.
   <pre>
    DB와 통신하기 위해 <code>JDBC API</code>를 사용하게되는데 My Batis는 JDBC에서의 번거로운작업을 대신해주고,
    XML 파일을 이용해 SQL 관리를 효율적으로 만듭니다.
